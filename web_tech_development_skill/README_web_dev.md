## My Library App

A web application to manage and explore your personal book collection. Users can add books, view, sort, and filter by title, author, year, genre, and rating. The app helps track favorite reads and explore different genres easily.

## Project: SmartHomeTech Website
SmartHomeTech is a responsive website showcasing a smart home technology company.  
The site presents information about the company, its products, technologies, benefits, news, and contact details.  
Designed to highlight usability, modern design, and interactive features such as a fixed sidebar and product cards.
### Skills
- Responsive Web Design
- Semantic HTML and CSS structuring
- Flexbox layout and grid systems
- Styling and hover effects for interactive UI
- Sidebar and navigation design
- Web content organization and accessibility
